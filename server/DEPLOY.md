## API

### `POST /api/install`  `{ "slug": "ci-analyst" }`
Increments the global tally **and** the current ISO-week bucket. Rejects any slug
that isn't `^[a-z0-9][a-z0-9-]{0,63}$` (so the slug can't smuggle data). Stores
only slugs + integer counts â€” no IP/UA/PII. Returns `{ ok, slug, count }`.

### `GET /api/install?weeks=8`
Read-only, CORS-open, cached `s-maxage=60`. Returns:

```json
{
  "ok": true,
  "installs": { "ci-analyst": 128, "content-marketer": 73 },
  "weekly": {
    "2026-W26": { "ci-analyst": 12, "content-marketer": 5 },
    "2026-W25": { "ci-analyst": 9 }
  },
  "weeks": ["2026-W19", "...", "2026-W26"]
}
```
